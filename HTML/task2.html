JavaScript:

uses three main keywords to declare variables:

Keyword Description
Let Modern way(block-scoped). Preferred for values that will change reassignable
const Constant(block-scoped). Preferred for values that won't change (but object properties can change)
var Old way (function-scoped). Avoid using in modern JS.

🡒 Example:

let age = 25; //Immutable

const PI = 3.14; //immutable (reassignment throws error)

var name = "Alice"; //Avoid (leaks scope)
